# 🍳 Love-Kitchen 爱心厨房

> AI 菜谱推荐小程序 - 帮你解决"今天吃什么"的世纪难题

---

## 这是干啥的？

一个会 AI 推荐菜谱的小程序，你告诉它冰箱里有什么食材，它就能帮你搭配出美味又营养的菜谱。

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 🥬 冰箱清库存 | 选择家里有的食材，AI 帮你搭配菜谱 |
| 🏃 跑者专属 | 针对跑步爱好者，智能推荐 3:1 碳水蛋白比恢复餐 |
| ⚡ 快手菜 | 15 分钟快手料理 |
| 💪 增肌餐 | 高蛋白健身餐 |
| ❤️ 收藏 | 喜欢的菜谱可以收藏起来 |
| 🔥 历史记录 | 记住你搜过什么 |

---

## ✨ 特色亮点

1. **AI 智能推荐** - 不是随机推荐，会根据你的食材、场景（跑步/增肌/快手）来定制
2. **场景化推荐** - 跑者、快手菜、增肌、日常四种模式
3. **本地缓存** - 联网生成过的菜谱会缓存，下次打开秒开
4. **零运营成本** - 用 DeepSeek API，按需付费

---

## 📱 适合谁用？

- 🏃 跑步爱好者（马拉松，月跑量 200+km）
- 💪 健身增肌人群
- 🤯 做饭纠结症患者
- 🧊 想清冰箱存货的人

---

## 🛠 技术栈

- **前端框架**: Taro (React)
- **AI 能力**: DeepSeek API
- **状态管理**: 本地缓存 + Storage
- **类型**: TypeScript
