# 🍳 爱心厨房

> 微信小程序 · AI 菜谱与冰箱助手 —— 帮家里「清库存、少浪费、吃好饭」

---

## 这是啥？

**爱心厨房**面向**负责一家人吃饭**的场景：记下冰箱里有什么、临期先吃、按食材出菜谱。不必先填满冰箱也能用——首页可直接输入食材搜索，或浏览今日推荐。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 🏠 清冰箱 | 双门冰箱视图管理库存，与「选菜」联动，临期优先 |
| 🍳 智能选菜 | 勾选/输入食材，匹配能消耗库存的菜谱 |
| 🏠 首页搜索 | 不依赖冰箱，输入食材即可 AI 推荐 |
| 🌤 今日推荐 | 默认参考天气；点「刷新天气」再尝试定位与实时天气（需授权） |
| ⚡ 场景 | 日常、跑后加餐、快手、增肌（跑后为可选场景，主路径仍是家庭日常） |
| ❤️ 收藏 | 菜谱可收藏，在「我的」中查看 |

---

## 技术栈

- **前端**：Taro + React + TypeScript + MobX  
- **AI**：MiniMax（OpenAI 兼容接口，默认模型 MiniMax-M2.7）；**生产环境推荐**服务端中转（`TARO_APP_LLM_PROXY_URL`），密钥不放客户端  

---

## 快速开始

```bash
npm install
npm run dev:weapp    # 微信小程序开发
npm run build:weapp
npm run build:h5
```

本地探活 MiniMax（需项目根 `.env.local` 中配置 `TARO_APP_MINIMAX_API_KEY`）：

```bash
npm run smoke:minimax
```
